# Currency Rates Parser (ЦБ РФ)

Многофункциональный консольный инструмент для получения и анализа курсов валют с сайта Центрального Банка России. Сохраняет данные в SQLite и предоставляет удобный CLI.

## Возможности

- Получение курсов на сегодня или на конкретную дату
- Поддержка нескольких валют: USD, EUR, CNY (легко добавить новые)
- Сохранение в SQLite с защитой от дубликатов
- **Умная история:** при запросе истории автоматически загружает недостающие даты из API
- Просмотр истории за любой период: дни, месяцы, годы
- Построение графиков курсов валют
- Подключение альтернативных источников данных (ЦБ РФ, ExchangeRate-API)

## Установка и запуск

```bash
git clone https://github.com/smhst/currency_parser.git
cd currency_parser
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

## Использование (CLI)
Обычный запуск (курсы на сегодня, все валюты)
```bash
python main.py
```
<!-- Курсы на 2026-05-29 (cbr): {'USD': 71.3715, 'EUR': 83.6892, 'CNY': 10.4487}
Курсы сохранены в базу данных. -->

Курсы на конкретную дату
```bash
python main.py --date 22.05.2026 --currencies USD EUR
```

История с автоматическим заполнением

Альтернативные API
Переключение на открытый ExchangeRate-API:
```bash
python main.py --api erapi --currencies USD
```
<!-- Курсы на 2026-05-28 (erapi): {'USD': 70.922}
Курсы сохранены в базу данных. -->

Визуализация
Построение графика курса валюты за указанный период:
```bash
python main.py --plot USD --days 30
```
