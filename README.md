# Currency Rates Parser (ЦБ РФ)

Многофункциональный консольный инструмент для получения и анализа курсов валют с сайта Центрального Банка России. Сохраняет данные в SQLite и предоставляет удобный CLI, графический интерфейс (Tkinter) и веб-интерфейс (Flask).

## Возможности

- Получение курсов на сегодня или на конкретную дату
- Поддержка динамического списка валют (добавление/удаление через CLI, GUI, веб)
- Сохранение в SQLite с защитой от дубликатов
- **Умная история:** при запросе истории автоматически загружает недостающие даты из API
- Просмотр истории за произвольный период, месяц или год
- Построение графиков курсов валют (сохраняются в PNG)
- Подключение альтернативных источников данных (ЦБ РФ, ExchangeRate-API)
- Три интерфейса: командная строка, десктопное окно, веб-сервер
- Упаковка в Docker для лёгкого развёртывания

## Установка и запуск

### Через Docker (только вэб-интерфейс)

git clone https://github.com/smhst/currency_parser.git
cd currency_parser
docker compose up -d

Веб-интерфейс будет доступен на http://localhost:5000. База данных сохраняется в локальной папке data/.

### Локально (без Docker)

```bash
git clone https://github.com/smhst/currency_parser.git
cd currency_parser
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

### Запуск GUI
```bash
python main.py --gui
```

### Запуск web
```bash
python web.py
```
Откройте в браузере http://127.0.0.1:5000.

### Использование (CLI)

Обычный запуск
```bash
python main.py
```

Курсы на конкретную дату
```bash
python main.py --date 22.05.2026 --currencies USD EUR
```

История

    Последние 30 дней: python main.py --history

    За период: python main.py --history --period 01.05.2026 10.05.2026

    За месяц: python main.py --history --month 2026-05

    За год: python main.py --history --year 2026

    Фильтр валют: python main.py --history --month 2026-05 --currencies USD CNY

Графики

    Последние 30 дней: python main.py --plot USD

    За месяц: python main.py --plot EUR --month 2026-05

    За год: python main.py --plot USD --year 2025

    Произвольный период: python main.py --plot CNY --period 01.04.2026 30.04.2026

График сохраняется в папку charts/.

Управление валютами
```bash
python main.py --add-currency JPY
python main.py --remove-currency JPY
```

Альтернативные API
```bash
python main.py --api erapi --currencies USD
```


```bash

```

График USD за весь 2025 год:
```bash

```

График за произвольный период:
```bash

```

График сохраняется в папку charts/ с уникальным именем (например, usd_30d_20260528.png).


Альтернативные API

Переключение на открытый ExchangeRate-API (только текущие курсы):
```bash
python main.py --api erapi --currencies USD
```

При попытке запросить историю с этим API произойдёт автоматическое переключение на ЦБ РФ с предупреждением.
